# 相关框架

# Sigma.js

Sigma.js (opens new window)是一个用于图绘制的 JavaScript 库，它支持通过简单的配置生成图，并可以方便进行拓展。 Sigma.js近乎停止维护 (opens new window)，同时在大规图数据下表现不佳。

# Stardust.js

Stardust.js (opens new window)是一个基于 WebGL 用于绘制信息可视化的 JavaScript 库。它通过WebGL加速，对于一般图形的的绘制性能表现优秀。Stardust.js 也可用来绘制简单的图数据。

# D3.js

D3.js (opens new window)是一个基于数据操作 DOM 的 JavaScript 库。它的设计精巧的Data-Driven的特性以及便捷的 DOM 操作 API 使它成为可视化可视分析领域中应用最广泛的工具。D3.js 可以和 NetV.js 结合使用，如使用 D3.js 进行布局结合用 NetV.js 进行渲染。

# Cytoscape.js

Cytoscape.js (opens new window)是一个面向图可视化、图可视分析的 JavaScript 库。它有丰富的图数据处理相关接口和完善的示例。
